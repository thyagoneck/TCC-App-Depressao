# HOPE
Esse app foi desenvolvido para o TCC.
Foi desenvolvido pensando em ser um tipo de rede social com foco em pessoas que desejam conversar com outras pessoas sobre seus problemas e conflitos pessoais, mantendo o seu anonimato. Foi pensado com o objetivo de ser uma ponte entre profissionais ou estudantes da psicologia, para adquirir experiÃªncia ao mesmo tempo que ajudam pessoas.

#### Tecnologias utilizadas

- Android Studio
- Firebase Database
- Firebase Cloud Functions
- Firebase Notification
- Firebase Cloud Messaging
- Astah
- Adobe XD
